
Exercise 16: 

Implementing Responsive Design with Media Queries

Description: 

Write a media query in your CSS that changes the 
background color of the <body> to white when the 
viewport width is less than 600px.

To test this, right click your Chrome web browser window when 
viewing the index.html file and select "Inspect". 

Then use the follow Chrome  dev tool feature linked below to 
shrink the screen size to see the change:
https://developer.chrome.com/docs/devtools/device-mode
